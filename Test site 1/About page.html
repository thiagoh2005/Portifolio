About | Pokemon Emerald RPG


Introduction

Pokemon Emerald RPG is a text-based fangame of pokemon made by thiagoh2005 as a way to learn HTML, CSS and JavaScript and have fun with it. In this game you go on an adventure to catch 'em all and become the greatest pokemon trainer, while dealing with some weird people, entities and "things" in the way...


How to play

It's easy. The game will display to you the following screens when playing:

Location, time of the day

Description of the place/situation *

Actions: *

1.Action 1 2.Action 2 3.Action 3

*1.try to pay attention to the description to think which action choose, before you lose something.
*Actions can be anything, i.e you're at a florest, and you see a suspicious pokemon, one of these actions can be something like: 1. Analyse wild pokemon. And with that you may learn which pokemon you're looking.
